# ctrl_art

ゲームパッド（またはキーボード）で VTuber 立ち絵をリアルタイム操作する Electron アプリ。  
OBS のウィンドウキャプチャ（透明背景）での使用を想定しています。

## 機能

- ゲームパッド／キーボードで立ち絵を移動・拡縮
- 表情・衣装・キャラクターの切替
- 口パクモード（マイク音量検出で口開け／口閉じ画像を自動切替）
- 表情キーで口パクを上書き、リセットキーで口パクモードに復帰
- フワフワアニメーション（sin 波）
- クリックスルーモード（透明ウィンドウを素通りして他のアプリを操作可能）
- 複数キャラクター・衣装・表情スロット対応
- ボタンマップ／キーボードマップを GUI で自由に変更可能
- 重複割り当て時に自動で競合を解除

## 必要環境

- Node.js 18 以上
- macOS または Windows 10/11
- ゲームパッド（Xbox 系推奨）※なしでもキーボードで操作可能

## セットアップ

```bash
npm install
npm start
```

## アセット（立ち絵画像）について

**このリポジトリには画像ファイルは含まれていません。**  
立ち絵画像は著作権の関係で各自ご用意ください。

設定画面（Start ボタンで開く）から登録できます：

- 表情スロットをクリック → ファイル選択ダイアログ
- 画像ファイルをスロットにドラッグ＆ドロップ

PNG を推奨（透過対応）。

### 口パク用画像について

衣装ごとに **口開け** と **口閉じ** の画像を設定できます。  
マイクが音を検出している間は口開け画像、無音時は口閉じ画像を表示します。  
表情キーを押すと口パクモードを上書きし、リセットキーで口パクモードに戻ります。

## デフォルト操作

### ゲームパッド（Xbox 系）

| 入力 | 機能 |
|---|---|
| A / B / X / Y / LB | 表情 1〜5 |
| RT | 口パクモードに戻る |
| RB | フワフワ ON/OFF |
| View（旧 Back） | 口パク ON/OFF |
| LT（長押し） | 位置リセット |
| Menu（旧 Start） | 設定を開く |
| 十字キー 左右 | 衣装切替 |
| 十字キー 上下 | キャラ切替 |
| 左スティック | 移動 |
| 右スティック Y 軸 | 拡縮 |

### キーボード（ゲームパッド未接続時のみ有効）

| キー | 機能 |
|---|---|
| 矢印キー | 移動 |
| `=` / `-` | 拡大 / 縮小 |
| 1〜5 | 表情 1〜5 |
| `` ` `` | 口パクモードに戻る |
| Q | フワフワ ON/OFF |
| E | 口パク ON/OFF |
| R（長押し） | 位置リセット |
| F | 設定を開く |
| `[` / `]` | 衣装切替 |
| `,` / `.` | キャラ切替 |

> ゲームパッドが接続されている間はキーボード操作は無効になります。

### クリックスルーモード

`Cmd+Shift+T`（Windows: `Ctrl+Shift+T`）で切替。  
ON にすると透明ウィンドウを素通りして他のアプリをそのまま操作できます。  
OBS キャプチャはクリックスルー中も通常通り 1920×1080 で取得できます。

| `npm run dist` | インストーラービルド |

**Windows**
```

```
